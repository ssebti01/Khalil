# Plan 07-01: Chicago & Houston Maps + Wind System

## Goal

Add Chicago Lakefront Park and Houston Rodeo Arena maps to `src/config/maps.js`. Implement a wind-force mechanic in `GameScene.js` that reads `map.windForce` and applies a periodic horizontal impulse to the ball — Chicago uses this mechanic. Houston has a high-restitution floor and a center launch ramp.

## Context

The map system (`src/config/maps.js` + `src/systems/MapLoader.js`) was introduced in Phase 4. By this phase the `MAPS` array contains: default stadium, Rabat, Bouskoura (Phase 5), Shanghai, NYC (Phase 6).

**Map schema (relevant new field):**

```js
{
  windForce: null | { x: Number, y: Number, intervalMs: Number }
  // If non-null, GameScene registers a repeating timer that applies
  // a temporary horizontal force to the ball every intervalMs ms.
}
```

**Wind system — how it must work in GameScene:**

`GameScene.create()` must call a new private method `_setupWind(mapConfig)` after the map is loaded. If `mapConfig.windForce` is non-null, it registers:

```js
_setupWind(mapConfig) {
  if (!mapConfig || !mapConfig.windForce) return;
  const wf = mapConfig.windForce;
  this._windActive = false;
  this._windTimer = this.time.addEvent({
    delay: wf.intervalMs,
    loop: true,
    callback: () => {
      this._windActive = true;
      // Apply for 500ms then stop
      this.time.delayedCall(500, () => { this._windActive = false; });
    },
  });
}
```

In `GameScene.update()`, when `this._windActive` is true, apply force each frame:

```js
if (this._windActive && this.ball && this.ball.body) {
  const wf = this._currentMap.windForce;
  Matter.Body.applyForce(
    this.ball.body,
    this.ball.body.position,
    { x: wf.x * 0.0001, y: 0 }
  );
}
```

On match end (`_endMatch`), call `this._windTimer?.remove()` to stop the wind timer. On pause (Phase 3 PauseScene), the scene pause will naturally stop `update()` so no extra handling needed.

**`this._currentMap`** must be set in `GameScene.init()` or `GameScene.create()` by looking up the map from the MAPS array by `mapId`. Import `{ MAPS }` from `'../config/maps.js'`.

**Arena constants:** Width 1280, Height 720, Floor y 640, Crowd zone y 560–640.

**Physics note for `floorRestitution`:** MapLoader applies this value when creating the floor body. Houston uses `floorRestitution: 0.45`. The existing default stadium floor uses `restitution: 0.2` (hardcoded in `_createWalls()`). MapLoader's floor body supersedes that for non-default maps — confirm MapLoader replaces the floor body correctly when a custom map is loaded.

### Task 2: Implement wind system in GameScene.js

**File:** `src/scenes/GameScene.js`

**Action:** Wire the wind mechanic into GameScene using the map config. Add `_setupWind()`, add `_windActive` flag checks in `update()`, and clean up timer in `_endMatch()`. Also import `MAPS` from `maps.js` and resolve `this._currentMap` in `init()`.

**Details:**

**Step A — Import MAPS and resolve current map in `init()`:**

Add to the top of `GameScene.js` (after existing imports):
```js

```

In `init(data)`, after existing assignments add:
```js
this.mapId = data.mapId ?? 'default';

```

**Step B — Call `_setupWind` in `create()`:**

At the end of `create()`, before the match timer block, add:
```js

```

**Step C — Add `_setupWind()` method:**

Add as a new private method (after `_drawArena` or before `update`):
```js
_setupWind(mapConfig) {
  if (!mapConfig || !mapConfig.windForce) return;
  const wf = mapConfig.windForce;
  this._windTimer = this.time.addEvent({
    delay: wf.intervalMs,
    loop: true,
    callback: () => {
      this._windActive = true;
      this.time.delayedCall(500, () => {
        this._windActive = false;
      });
    },
  });
}
```

**Step D — Apply wind force in `update()`:**

In `update(time, delta)`, after the `if (this.matchOver) return;` guard, add:
```js
// Wind force application (Chicago map mechanic)
if (this._windActive && this._currentMap?.windForce && this.ball?.body) {
  const wf = this._currentMap.windForce;
  this.matter.world.engine.world; // ensure engine available
  Matter.Body.applyForce(
    this.ball.body,
    this.ball.body.position,
    { x: wf.x * 0.0001, y: 0 }
  );
}
```

Note: `Matter` is available globally via Phaser's bundled Matter.js. If not auto-global, import via:
```js
import Phaser from 'phaser';
const Matter = Phaser.Physics.Matter.Matter;
```
Use `this.matter.world.engine` to access the Matter engine if direct Matter reference is unavailable. The force scalar `0.0001` converts the config's px/frame unit into a Matter force magnitude appropriate for a ball mass of ~0.8.

**Step E — Clean up timer in `_endMatch()`:**

At the start of `_endMatch()` (before setting `this.matchOver = true`):
```js
if (this._windTimer) {
  this._windTimer.remove(false);
  this._windTimer = null;
}
this._windActive = false;
```
