# Plan 01-01: Fix Goal Physics (Ball Stops at Goal)

## Goal

The ball currently stops dead or bounces back when it reaches the goal mouth instead of rolling through to score. Fix the geometry so the ball fits cleanly inside the goal and the score sensor fires reliably.

## Context

Two compounding geometry bugs in `GameScene.js` prevent goals from registering:

**Bug A — Back wall too close.** `_createWalls()` places the left goal back wall at center `x = -thick/2 = -20` (with `thick = 40`). Its right face sits at `x = 0`. The goal post left face sits at `x = 57` (post center 62, half-width 5). The gap is only 57px. The ball diameter is 52px (`BALL.radius = 26`). The ball barely fits and the restitution-0.4 back wall bounces it right back out before the sensor fires.

**Bug B — Score sensor fires too early.** `_createGoals()` places `leftGoalZone` at center `x = 30`, half-width 25, so it spans `x = 5..55`. The goal post face is at `x = 57`. The sensor overlaps open field — the ball triggers it before it has passed the post. The 2500ms cooldown (`MATCH.goalCooldown`) then blocks any re-trigger, so a valid shot immediately after is ignored.

The right goal mirrors all of these problems symmetrically.

## Must Haves

- [ ] Kicking the ball into the left goal from the right side of the pitch scores a point for Player 2
- [ ] Kicking the ball into the right goal from the left side of the pitch scores a point for Player 1
- [ ] The ball rolls through the goal mouth and comes to rest against the back wall (no bounce-back on a straight centre shot)
- [ ] The goal sensor does not trigger when the ball is still in front of the post

### Task 2: Reposition goal score sensors behind the post face

**File:** `src/scenes/GameScene.js`

**Action:** In `_createGoals()`, move both sensor rectangles so they sit fully inside the goal mouth — entirely behind the post face — eliminating false triggers from open-field contact.

**Details:**

Current `leftGoalZone` sensor (lines 172-175):
- center x = 30, half-width = 25 → spans x = 5..55
- Fires when ball is still in open field (post face is at x = 57)

Replace with center x = 10, half-width = 20 → spans x = -10..30. The sensor right edge (x=30) is less than post_face - ball_radius (57 - 26 = 31), guaranteeing the ball has physically passed the post before the sensor fires.

Current code:
```javascript
this.leftGoalZone = this.matter.add.rectangle(
  30, h - 80 - GOAL.height / 2, 50, GOAL.height,
  { isStatic: true, isSensor: true, label: 'goalLeft' }
);
```
Change to:
```javascript
this.leftGoalZone = this.matter.add.rectangle(
  10, h - 80 - GOAL.height / 2, 40, GOAL.height,
  { isStatic: true, isSensor: true, label: 'goalLeft' }
);
```

Geometry check: sensor right edge = 10 + 20 = 30. Post face = 57. Ball clears post when ball center > 57 + 26 = 83. Sensor fires when ball center > 30 + 26 = 56. Wait — sensor fires at ball center 56, post face is 57. The ball edge reaches the sensor (x=30) when ball center is at 56, which is just before the post face at 57. We need the sensor right edge < post_face - ball_radius = 31. With right edge at 30, sensor fires at ball center = 56, which is 1px before the post face. Use right edge = 20 to be safe: center x = 0, half-width = 20 → spans x = -20..20. Sensor fires when ball center > 20 + 26 = 46 — clearly inside the goal.

**Corrected final values:**
```javascript
this.leftGoalZone = this.matter.add.rectangle(
  0, h - 80 - GOAL.height / 2, 40, GOAL.height,
  { isStatic: true, isSensor: true, label: 'goalLeft' }
);
```

Current `rightGoalZone` (lines 176-179):
- center x = GAME_WIDTH - 30, half-width = 25 → spans x = 1225..1275
- Fires when ball is still in open field (right post face is at GAME_WIDTH - 57 = 1223)

Replace with center x = GAME_WIDTH, half-width = 20 → spans x = 1260..1300. Sensor fires when ball center < GAME_WIDTH - 26 = 1254 — ball is inside the goal with margin.

**Corrected final values:**
```javascript
this.rightGoalZone = this.matter.add.rectangle(
  GAME_WIDTH, h - 80 - GOAL.height / 2, 40, GOAL.height,
  { isStatic: true, isSensor: true, label: 'goalRight' }
);
```
