# Plan 04-02: Map Selector UI in MenuScene

## Goal

Add a horizontal map selector widget to `MenuScene` so players can pick a map before kick-off. The selected `mapId` is forwarded to `GameScene` via `scene.start()` init data.

This plan depends on 04-01 (requires `getMaps()` from `src/config/maps.js`).

## Context

**Relevant source (read before editing):**
- `src/scenes/MenuScene.js` — current layout:
  - `_drawBackground()` — sky gradient, ground, stars
  - `_drawTitle()` — "HEAD SOCCER" text at y=70
  - `_drawModeToggle()` — mode toggle text at y=155
  - `_drawCharacterSelect()` — two character cards centered at y=320 (cards are 260px tall, so bottom edge ~y=450)
  - `_drawStartButton()` — KICK OFF! button at y=570
  - `_drawControls()` — control hints at y=650
  - Constructor sets `this.vsMode`, `this.p1CharIndex`, `this.p2CharIndex`
  - `_startGame()` calls `this.scene.start('GameScene', { p1CharId, p2CharId, vsMode })`

**Layout constraint:** Map selector must fit between character cards (bottom ~y=460) and KICK OFF! button (y=570). Target center y=510.

**Dependency:** `getMaps()` from `src/config/maps.js` (created in plan 04-01) returns the full MAPS array. At plan 04-01 completion, only `stadium` exists. At plan 05-01 completion, `rabat` and `bouskoura` will be in the array — the selector will surface them automatically.

## Tasks

### Task 1: Add `mapIndex` state and `_drawMapSelector()` to MenuScene

**File:** `src/scenes/MenuScene.js`

**Action:** Four targeted changes:

**1. Add import** at top of file (alongside existing imports):
```javascript
import { getMaps } from '../config/maps.js';
```

**2. Initialize `mapIndex` in constructor** — add one line to the constructor body:
```javascript
this.mapIndex = 0;
```

**3. Add `_drawMapSelector()` method** to the class. Insert after `_drawCharacterSelect()` and before `_changeChar()`:

```javascript
_drawMapSelector() {
  const maps = getMaps();
  const cx = GAME_WIDTH / 2;
  const cy = 510;

  // Background strip
  const bg = this.add.graphics();
  bg.fillStyle(0x000000, 0.35);
  bg.fillRoundedRect(cx - 200, cy - 28, 400, 56, 10);
  bg.lineStyle(2, 0x888888, 0.6);
  bg.strokeRoundedRect(cx - 200, cy - 28, 400, 56, 10);

  // Label above
  this.add.text(cx, cy - 40, 'MAP', {
    fontSize: '14px', fontFamily: 'Arial Black, sans-serif',
    color: '#888888',
  }).setOrigin(0.5);

  // Map name text
  this._mapNameText = this.add.text(cx, cy, maps[this.mapIndex].name, {
    fontSize: '26px', fontFamily: 'Arial Black, sans-serif',
    color: '#ffffff',
  }).setOrigin(0.5);

  // Left arrow
  const leftArrow = this.add.text(cx - 170, cy, '◄', {
    fontSize: '28px', color: '#ffffff',
  }).setOrigin(0.5).setInteractive({ useHandCursor: true });
  leftArrow.on('pointerdown', () => {
    this.mapIndex = (this.mapIndex - 1 + maps.length) % maps.length;
    this._mapNameText.setText(maps[this.mapIndex].name);
  });
  leftArrow.on('pointerover', () => leftArrow.setColor('#ffee00'));
  leftArrow.on('pointerout', () => leftArrow.setColor('#ffffff'));

  // Right arrow
  const rightArrow = this.add.text(cx + 170, cy, '►', {
    fontSize: '28px', color: '#ffffff',
  }).setOrigin(0.5).setInteractive({ useHandCursor: true });
  rightArrow.on('pointerdown', () => {
    this.mapIndex = (this.mapIndex + 1) % maps.length;
    this._mapNameText.setText(maps[this.mapIndex].name);
  });
  rightArrow.on('pointerover', () => rightArrow.setColor('#ffee00'));
  rightArrow.on('pointerout', () => rightArrow.setColor('#ffffff'));
}
```

**Note on `scene.restart()` in `_updateCharLabels()`:** Phaser's `scene.restart()` re-calls `create()` but preserves the scene object's property values that were set on `this` before restart. `this.mapIndex` set in the constructor persists across restarts — no extra handling needed.

**4. Call `_drawMapSelector()` from `create()`** — insert the call in the `create()` method body, between `this._drawCharacterSelect()` and `this._drawStartButton()`:
```javascript
this._drawMapSelector();
```

**Verify:** `npm run dev` — MenuScene shows MAP label and map name widget between character cards and KICK OFF button. Clicking arrows cycles map name. No console errors.

**Done:** Map selector widget visible and interactive in MenuScene.

### Task 2: Pass `mapId` to GameScene on start

**File:** `src/scenes/MenuScene.js`

**Action:** Update `_startGame()` to include `mapId` in the scene data. Replace the existing `_startGame()` method body:

```javascript
_startGame() {
  const maps = getMaps();
  this.scene.start('GameScene', {
    p1CharId: CHARACTERS[this.p1CharIndex].id,
    p2CharId: CHARACTERS[this.p2CharIndex].id,
    vsMode: this.vsMode,
    mapId: maps[this.mapIndex].id,
  });
}
```
